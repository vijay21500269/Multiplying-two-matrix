## ALGORITHM:

### Step 2:
use append() to add new elements in the list

### Step 3:
Get the input from the user using np.array()

### Step 4:
result = value1*value2 it will multiply the array and print the output

### Step 5:
End the program

## PROGRAM: 
~~~
Developed By: R.Vijay
Ref no: 21500269
import numpy as np
z,y=[],[]
a=int(input())
for i in range(a):
    z.append(int(input()))
for i in range(a):
    y.append(int(input()))
var1=np.array(z)
var2=np.array(y)
res=var1*var2
print("Product of two arrays is:",res)
